# Projeto QA - OrangeHRM

Projeto desenvolvido para prática de testes funcionais manuais utilizando o sistema OrangeHRM Demo.

## Objetivo

Aplicar conceitos de Quality Assurance (QA) através da criação e execução de casos de teste funcionais em um sistema web corporativo.

## Funcionalidades testadas

- Login válido
- Login inválido
- Validação de campos obrigatórios
- Logout do sistema

## Atividades realizadas

- Criação de casos de teste
- Execução manual dos testes
- Registro de evidências
- Validação de comportamento do sistema
- Testes positivos e negativos

## Ferramentas utilizadas

- Excel
- GitHub
- OrangeHRM Demo
